## Adding a New Client

### Method 1: Manual Entry

1. Go to **Clients > New Client**
2. Fill in required fields:
   - **Name:** Client or company name
   - **Email:** For sending invoices
   - **Phone:** Contact number (optional)
3. Add address details:
   - Street address
   - City
   - State/County
   - Postal code
   - Country
4. Add business details:
   - Company name (if different)
   - Tax ID/VAT number
   - Website (optional)
5. Set preferences:
   - Payment terms (Net 15, Net 30, etc.)
   - Preferred payment method
   - CIS settings (if applicable)
   - Add notes (internal use)
6. Save client

### Method 2: CSV Import

1. Go to **Clients > Import Clients**
2. Download template CSV
3. Fill in client data
4. Upload CSV file
5. Review and confirm import

## Client Information Fields

### Required Fields

- **Name:** Client or company name
- **Email:** For invoice delivery

### Optional Fields

- Phone number
- Full address
- Company name
- Tax ID/VAT number
- Website
- Notes
- Payment preferences

## Editing Clients

1. Go to **Clients** page
2. Click on the three dots on the row for the client you want to edit
3. Click **Manage Client** button
4. Update information
5. Save changes

## Client Status

### Active Clients

- Default status
- Can create invoices
- Appears in dropdowns
- Included in reports

### Inactive Clients

- Archived but not deleted
- Hidden from dropdowns
- Invoices still accessible
- Can reactivate later

## Organizing Clients

### Search and Filter

- Search by name
- Filter by status
- Find quickly
- Sort alphabetically

### Client Notes

- Internal notes
- Payment history
- Special instructions
- Relationship details

## Best Practices

### Client Information

- Keep details up to date
- Include full addresses
- Add tax IDs when needed
- Note payment preferences

### Organization

- Use consistent naming
- Add clear notes
- Group similar clients
- Archive inactive ones

### Payment Terms

- Set client-specific terms
- Match their preferences
- Default to your standard
- Adjust as needed

Well-organised clients make invoicing faster!
