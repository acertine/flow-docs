## Business Information

### Required Fields

- **Business Name:** Your company name
- **Business Address:** Complete address
- **Tax ID/VAT Number:** If applicable

### Optional Fields

- **Logo:** Upload company logo
- **Website:** Business website
- **Phone:** Business phone

## Address Details

### Full Address

- Street address
- City
- State/County
- Postal code
- Country

### Address Settings

- Include on invoices (default: yes)
- Include phone number (default: yes)
- Hide if not needed

## Logo Upload

### Requirements

- High-quality image
- PNG or SVG format
- Transparent background recommended
- Appropriate size

### Logo Usage

- Appears on invoices
- Professional branding
- Can be updated anytime
- Maintains aspect ratio

## Best Practices

- Use complete business name
- Include full address
- Add VAT number if registered
- Use professional logo
- Keep information current

Professional business details build client trust!
