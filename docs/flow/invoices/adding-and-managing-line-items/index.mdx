## Adding Line Items

### Basic Line Item

1. Click **Add Item** button
2. Enter:

- Description: What you're charging for
- Quantity: Number of units
- Unit Price: Price per unit
- VAT Rate: Select applicable VAT rate

3. Total calculates automatically

### Multiple Line Items

- Click Add Item multiple times
- Each item calculates independently

## Line Item Tips

### Clear Descriptions

- Be specific: Web Development - Homepage Design vs Design
- Include details: 10 hours @ Â£50/hour
- Mention deliverables if applicable

### Quantity and Pricing

- Use appropriate units: hours, items, days, etc.
- Set realistic quantities
- Round prices appropriately

### VAT Handling

- Select correct VAT rate (0%, 5%, 20%, Exempt and Not Applicable)
- Choose VAT type: Inclusive or Exclusive
- VAT calculates automatically

## Editing Line Items

- Edit: Click the item to modify
- Delete: Click trash icon

## CIS Categories (Construction Industry Scheme)

If CIS is enabled:

- Labor: Construction work and labor
- Materials: Materials and supplies
- Other: Miscellaneous items

<Note type="note" title="Note">
  CIS deduction applies only to labor items.
</Note>

## Bulk Operations

- Duplicate entire invoice
- Copy line items between invoices
- Import from templates

## Best Practices

- Group related items together
- Use consistent naming conventions
- Include detailed descriptions
- Set appropriate VAT rates
- Review totals before sending

Well-structured line items make your invoices professional and easy to understand!
