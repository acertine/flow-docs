## Creating a Template

### From Existing Invoice

1. Open any invoice
2. Click **Save as Template**
3. Give it a name
4. Template saved for reuse

### From Scratch

1. Create new invoice
2. Set up structure you'll reuse
3. Add typical line items
4. Save as template

## Using Templates

1. Go to **Invoices > Templates**
2. Click on a template
3. Fill in client-specific details
4. Adjust amounts as needed
5. Save or send

## Template Features

### What Gets Saved

- Line item structure
- Descriptions and categories
- VAT rates
- Payment terms
- Notes and instructions
- CIS settings (if applicable)

### What Changes Each Time

- Client information
- Invoice number
- Dates
- Quantities and amounts
- Totals

## Managing Templates

### Edit Template

- Modify structure
- Update descriptions
- Change default values
- Save changes

### Delete Template

- Remove unused templates
- Clean up old structures
- Keep relevant ones

## Best Practices

### When to Use Templates

- Recurring services
- Standard service packages
- Common project types
- Regular retainer work

### Template Organisation

- Name templates clearly
- Use descriptive names
- Group by service type
- Keep templates updated

### Template Tips

- Use placeholders for variable amounts
- Include common notes
- Set standard payment terms
- Include typical line items

## Common Template Examples

### Hourly Work Template

- Line items for hours
- Standard hourly rate
- VAT included
- Clear descriptions

### Project-Based Template

- Fixed project fee
- Milestone breakdown
- Payment schedule notes
- Deliverables listed

### Retainer Template

- Monthly retainer amount
- Included services
- Additional work rates
- Terms and conditions

Templates streamline your invoicing workflow significantly!
