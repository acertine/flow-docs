## What is Stripe Connect?

Stripe Connect allows:

- Online payment processing
- Secure payment links
- Instant payment confirmation
- Automatic invoice updates
- Multiple payment methods

## Setting Up Stripe Connect

### Step 1: Get Stripe Account

1. Sign up at https://stripe.com
2. Complete account setup
3. Verify business details
4. Get account approved

### Step 2: Connect in Flow

1. Go to **Settings > Banking Information**
2. Click **Edit Preferences**
3. Click **Connect with Stripe**
4. Authorise Acertine Flow access
5. Complete connection

### Step 3: Configure Settings

- Enable payment links
- Set default currency
- Configure payment methods
- Test connection

## How Payment Links Work

### Automatic Links

- Created for each invoice
- Included in email
- Secure payment page
- Multiple payment methods

### Payment Process

1. Client clicks payment link
2. Enters payment details
3. Completes payment
4. Receives confirmation
5. Invoice marked as paid

### Payment Confirmation

- Automatic status update
- Email notification sent
- Receipt generated
- Payment recorded

## Payment Methods Supported
