## Supported Browsers

### Recommended

- Chrome (latest version)
- Firefox (latest version)
- Safari (latest version)
- Edge (latest version)
- Any recent Chromium variant (Helium, Vivaldi, Arc, etc)
- Any recent Firefox variant (Zen, etc)

### Minimum Versions

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## Browser Features Required

- JavaScript enabled
- Cookies enabled
- Local storage
- Modern CSS support

## Mobile Browsers

### Supported

- iOS Safari
- Chrome Mobile
- Firefox Mobile
- Samsung Internet

### Mobile Features

- Responsive design
- Touch-friendly interface
- Mobile-optimised forms
- Fast loading

## Troubleshooting Browser Issues

### Clear Cache

- Clear browser cache
- Clear cookies
- Hard refresh (Ctrl+F5 / Cmd+Shift+R)
- Try incognito mode

### Update Browser

- Install latest version
- Enable auto-updates
- Check for updates
- Restart browser

### Disable Extensions

- Temporarily disable extensions
- Check for conflicts
- Test in incognito
- Re-enable one by one

## Best Practices

- Use latest browser version
- Keep browser updated
- Clear cache regularly
- Disable problematic extensions

Using a modern browser ensures the best experience!
