## What Are Quotes?

Quotes are estimates or proposals sent to clients before work begins:

- Project estimates
- Service proposals
- Price quotations
- Budget outlines

## Creating a Quote

### Step 1: Create Base Quote

1. Create quote as normal
2. Fill in all details
3. Set up line items
4. Configure payment terms

### Step 2: Set Quote Details

1. Toggle Make this a quote
2. Set expiry date (optional)
3. Add custom message
4. Include terms and conditions

### Step 3: Review and Save

- Review all line items
- Check calculations
- Verify client information
- Save quote

## Sending Quotes

### Method 1: Send Immediately

1. Click Save & Send
2. Quote is emailed automatically
3. Client receives quote link

### Method 2: Send Later

1. Save quote as draft
2. Go to Quotes page
3. Find your quote
4. Click Send button

## Quote Features

### What Gets Sent

- Quote preview link
- PDF download option
- Expiry date (if set)
- Your custom message
- Terms and conditions

### Quote Status

- **Draft:** Not sent yet
- **Sent:** Emailed to client
- **Viewed:** Client opened the quote
- **Accepted:** Client approved
- **Converted:** Turned into invoice
- **Expired:** Past expiry date

## Converting Quotes to Invoices

### When Quote is Accepted

1. Open the quote
2. Click Convert to Invoice
3. Quote details copied to invoice
4. Adjust amounts if needed
5. Send invoice

### What Gets Converted

- All line items
- Client information
- Descriptions and amounts
- Payment terms
- Notes and instructions

## Managing Quotes

### View All Quotes

1. Go to Quotes page
2. Filter by status
3. View quote history
4. Check acceptance status

### Edit Quote

- Modify line items
- Update amounts
- Change expiry date
- Adjust terms

### Delete Quote

- Remove unused quotes
- Clean up old quotes
- Keep accepted ones

## Best Practices

### Creating Effective Quotes

- Be clear and detailed
- Include all relevant items
- Set realistic expiry dates
- Use professional language
- Include terms and conditions

### Following Up

- Track quote views
- Follow up on sent quotes
- Remind before expiry
- Convert accepted quotes promptly

### Quote Organisation

- Use clear descriptions
- Name quotes consistently
- Group by client or project
- Archive old quotes

## Quote Templates

You can also create quote templates:

- Save common quote structures
- Reuse for similar projects
- Maintain consistency
- Save time on setup

Quotes help you present professional proposals and win more business!
